# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-13

### Added
- Initial release of Free LLM Router n8n node
- LangChain-compatible model interface
- Support for Smart Strategy, specific models, and priority lists
- Advanced filtering options (tags, type, context size, success rate)
- Support for all OpenAI-compatible parameters
- Authentication support (None, Basic Auth, Bearer Token)
- Comprehensive documentation and examples
